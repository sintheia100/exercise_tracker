API Project: Exercise tracker

Create a New User

POST /api/users

Add exercises

POST /api/users/:_id/exercises

Exercise Logs

GET /api/users/:_id/logs?[from][&to][&limit]

GET user's exercise log: GET /api/users/:_id/logs?[from][&to][&limit]

[ ] = optional

from, to = dates (yyyy-mm-dd); limit = number
